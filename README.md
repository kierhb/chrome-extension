## Introduction
- This is my solution to the third and final part of [Learn JavaScript](https://scrimba.com/learn/learnjavascript).
- This is a chrome extension, with some functionalities such as:
    - Save Input Button
    - Save Tab Button
    - Delete All Button
    - Accessing local storage
